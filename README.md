# E-commerce Sales Dashboard (Excel)

## Overview

#### 1.This project is an **E-commerce Annual Report Dashboard (2022)** created in **Microsoft Excel**.
#### 2.It helps analyze sales, orders, customer demographics, and sales channels using interactive charts and filters.

## Tools Used

##### 1.Microsoft Excel.
#### 2.Pivot Tables.
#### 3.Pivot Charts.
#### 4.Slicers.

## Dashboard Insights

#### 1."Orders vs Sales" by month.
#### 2."Men vs Women" sales comparison.
#### 3."Order Status" (Delivered, Returned, Cancelled, Refunded).
#### 4."Top 10 States by Sales".
#### 5."Orders by Age & Gender".
#### 6."Channel-wise Sales" (Amazon, Flipkart, Myntra, Ajio, etc.).

## Filters Available

#### 1.Month
#### 2.Sales Channel
##### 3.Product Category

#### These filters make the dashboard **interactive and easy to analyze**.

## File

#### `ecommerce_dashboard.xlsx` – Excel file containing the full dashboard.

## Purpose

#### The goal of this project is to practice **data analysis and dashboard creation in Excel** to understand business performance and customer trends.

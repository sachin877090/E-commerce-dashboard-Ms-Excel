# E-commerce Sales Dashboard (Excel)

## Overview

This project is an **E-commerce Annual Report Dashboard (2022)** created in **Microsoft Excel**.
It helps analyze sales, orders, customer demographics, and sales channels using interactive charts and filters.

## Tools Used

>> Microsoft Excel
>> Pivot Tables
>> Pivot Charts
>> Slicers

## Dashboard Insights

The dashboard shows:

>> "Orders vs Sales" by month
>> "Men vs Women" sales comparison
>> "Order Status" (Delivered, Returned, Cancelled, Refunded)
>> "Top 10 States by Sales"
>> "Orders by Age & Gender"
>> "Channel-wise Sales" (Amazon, Flipkart, Myntra, Ajio, etc.)

## Filters Available

> Month
> Sales Channel
> Product Category

These filters make the dashboard **interactive and easy to analyze**.

## File

`ecommerce_dashboard.xlsx` – Excel file containing the full dashboard.

## Purpose

The goal of this project is to practice **data analysis and dashboard creation in Excel** to understand business performance and customer trends.
